Java集合之LinkedList

发表于2021-06-27|更新于2021-08-02|Java小知识点
|字数总计:1.3k|阅读时长:5分钟|阅读量:

类结构

源码分析

底层实现

LinkedList的底层存储结构是双向链表,对于频繁的插入删除操作效率要比ArrayList高

private static class Node<E> {
    E item;
    Node<E> next;
    Node<E> prev;
    Node(Node<E> prev, E element, Node<E> next) {
        this.item = element;
        this.next = next;
        this.prev = prev;
    }
}

和ArrayList不同的是:ArrayList是用Object[]存储元素,而ArrayList则是用Node来代表一个元素,next和prev则是代表这个元素的上一个元素和后一个元素

构造方法

  • 空参构造

    public LinkedList() {
}

    这个没啥好说的…

  • 有参构造

    public LinkedList(Collection<? extends E> c) {
    this();
    addAll(c);
}
    public boolean addAll(Collection<? extends E> c) {
    return addAll(size, c);
}
    public boolean addAll(int index, Collection<? extends E> c) {
    checkPositionIndex(index);
    Object[] a = c.toArray();
    int numNew = a.length;
    if (numNew == 0)
        return false;
    Node<E> pred, succ;
    if (index == size) {
        succ = null;
        pred = last;
    } else {
        succ = node(index);
        pred = succ.prev;
    }
    for (Object o : a) {
        @SuppressWarnings("unchecked") E e = (E) o;
        Node<E> newNode = new Node<>(pred, e, null);
        if (pred == null)
            first = newNode;
        else
            pred.next = newNode;
        pred = newNode;
    }
    if (succ == null) {
        last = pred;
    } else {
        pred.next = succ;
        succ.prev = pred;
    }
    size += numNew;
    modCount++;
    return true;
}
    private void checkPositionIndex(int index) {
    if (!isPositionIndex(index))
        throw new IndexOutOfBoundsException(outOfBoundsMsg(index));
}
    private boolean isPositionIndex(int index) {
    return index >= 0 && index <= size;
}

    分析:

    1. 首先调用空参构造方法创建一个LinkedList对象
    2. 然后调用了addAll()方法,这里我们直接进到最底层开始分析,先通过isPositionIndex(int index)方法判断下标是否越界(因为是创建对象所以这里的index和size都是0)
    3. 判断完下标是否越界后,将参数Collection转换为数组并遍历,将数组中的元素作为节点按顺序连接起来
    4. 修改LinkedList的长度size

添加节点

public boolean add(E e) {
    linkLast(e);
    return true;
}
void linkLast(E e) {
    final Node<E> l = last;
    final Node<E> newNode = new Node<>(l, e, null);
    last = newNode;
    if (l == null)
        first = newNode;
    else
        l.next = newNode;
    size++;
    modCount++;
}
Node(Node<E> prev, E element, Node<E> next) {
    this.item = element;
    this.next = next;
    this.prev = prev;
}

分析:

  1. 先来分析linkLast(),先创建一个局部变量l作为最后一个节点

  2. 然后利用Node的有参构造将添加的元素作为节点创建出来,并连接上最后一个节点

  3. 将LinkedList最后一个节点last指向这个刚刚创建的节点

  4. 判断LinkedList的最后一个节点l是否为null:

    • 如果是就代表这是第一次添加节点,将新创建的节点作为LinkedList的首节点first;
    • 如果不是将之前的最后一个节点和这个新创建的节点连接起来l.next;

  5. 最后长度+1

删除节点

  • 无参

    删除第一个节点

    public E remove() {
    return removeFirst();
}
    public E removeFirst() {
    final Node<E> f = first;
    if (f == null)
        throw new NoSuchElementException();
    return unlinkFirst(f);
}
    private E unlinkFirst(Node<E> f) {
    // assert f == first && f != null;
    final E element = f.item;
    final Node<E> next = f.next;
    f.item = null;
    f.next = null; // help GC
    first = next;
    if (next == null)
        last = null;
    else
        next.prev = null;
    size--;
    modCount++;
    return element;
}

    分析:

    1. 先判断第一个节点是否为null如果是抛出NoSuchElementException异常
    2. 然后将节点内容和节点指向的下一个节点分别赋值给局部变量item和next
    3. 将删除的节点的内容和下一个节点赋值为null
    4. 因为删除的是第一个节点,所以下一个节点也就是第二个节点成了第一个节点
    5. 判断原来第二个节点是否为null:
      • 是,最后一个节点也为null这个LinkedList就是null
      • 否,这个节点作为首节点,上一个节点prev就是null
    6. 然后长度-1

  • 有参remove(int)

    public E remove(int index) {
    checkElementIndex(index);
    return unlink(node(index));
}
    Node<E> node(int index) {
    // assert isElementIndex(index);
    if (index < (size >> 1)) {
        Node<E> x = first;
        for (int i = 0; i < index; i++)
            x = x.next;
        return x;
    } else {
        Node<E> x = last;
        for (int i = size - 1; i > index; i--)
            x = x.prev;
        return x;
    }
}
    E unlink(Node<E> x) {
    // assert x != null;
    final E element = x.item;
    final Node<E> next = x.next;
    final Node<E> prev = x.prev;
    if (prev == null) {
        first = next;
    } else {
        prev.next = next;
        x.prev = null;
    }
    if (next == null) {
        last = prev;
    } else {
        next.prev = prev;
        x.next = null;
    }
    x.item = null;
    size--;
    modCount++;
    return element;
}

    分析:

    1. 首先判断参数index是否越界

    2. 然后根据参数遍历出要删除的节点

    3. 将要删除的节点的内容,prev,next赋值给局部变量

    4. 判断prev是否为null:

      • 如果是那么要删除的节点就是首节点,将第二个节点设置为首节点

      • 如果不是,那么将上一个节点的next指向下一个节点,要删除的节点的上一个节点设置为null:

    5. 判断next是否为null:

      • 是,那么就代表要删除的节点是最后一个节点,则需要把倒数第二个节点设置为最后一个节点
      • 否,那么将要删除的节点的下一个节点的prev指向要删除节点的prev(上一个节点),要删除的节点的next设置为null

    6. 要删除的节点的内容设置为null

    7. 长度-1
