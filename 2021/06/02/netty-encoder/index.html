编写客户端

因为项目是作为客户端的所以这里只写客户端了

@Slf4j
@Component
public class NettyClient {
    @Autowired
    private NettyProperties nettyProperties;
    @Autowired
    private NettyServerInitializer nettyServerInitializer;
    public void nettyClientStart() {
        EventLoopGroup clientGroup = new NioEventLoopGroup();
        Bootstrap clientBootStrap = new Bootstrap();
        try {
            clientBootStrap.group(clientGroup)
                    .channel(NioSocketChannel.class)
                    .handler(nettyServerInitializer);
            ChannelFuture channelFuture = clientBootStrap.connect(nettyProperties.getIp(), nettyProperties.getPort()).sync();
            if (channelFuture.isSuccess()) {
                log.info("连接服务端成功");
            }else {
                log.error("连接服务端失败");
            }
            channelFuture.channel().closeFuture().sync();
        } catch (InterruptedException e) {
            e.printStackTrace();
        } finally {
            clientGroup.shutdownGracefully();
        }
    }
}

netty初始化类

@Component
@ChannelHandler.Sharable
public class NettyServerInitializer extends ChannelInitializer<SocketChannel> {
    @Autowired
    private NettyServerHandler nettyServerHandler;
    private static NettyEncoder nettyEncoder = new NettyEncoder();
    @Override
    protected void initChannel(SocketChannel socketChannel) throws Exception {
        ChannelPipeline pipeline = socketChannel.pipeline();
        // 使用自定义的编码器
        pipeline.addLast(nettyEncoder);
        pipeline.addLast(nettyServerHandler);
    }
}

编写编码器

注意:这里是编码器的核心代码 
@Component
@ChannelHandler.Sharable
public class NettyEncoder extends MessageToByteEncoder<String> {
    @Override
    protected void encode(ChannelHandlerContext channelHandlerContext, String s, ByteBuf byteBuf) throws Exception {
        Channel channel = channelHandlerContext.channel();
        // 将字符串转换为16进制发送给服务端
        channel.writeAndFlush(Unpooled.copiedBuffer(HexBinaryUtil.hexStringToByteArray(s)));
    }
}

事件处理类

我这里是读取完消息直接反馈给服务端数据包,没有做任何业务处理

public void channelRead(ChannelHandlerContext ctx, Object msg) throws Exception {
    ByteBuf byteBuf = (ByteBuf) msg;
    byte[] bytes = new byte[byteBuf.readableBytes()];
    byteBuf.readBytes(bytes);
    String clientMsg = HexBinaryUtil.bytesToString(bytes);
    log.info("收到 ip = {} 发来的消息 msg = {}", ctx.channel().remoteAddress(), clientMsg);
    Channel channel = ctx.channel();
    // 发送消息: "11050000ff008eaa"
    ChannelFuture channelFuture = channel.writeAndFlush(ElevatorCommand.CLOSE_STEP_1);
    channelFuture.addListener(new ChannelFutureListener() {
        @Override
        public void operationComplete(ChannelFuture channelFuture) throws Exception {
            if (channelFuture.isDone()) {
                if (channelFuture.isSuccess()) {
                    log.info("================== 发送完成 =======================");
                }
            }else {
                log.error("================== 发送失败 =======================");
            }
        }
    });
}

    使用到的两个进制转换方法

    16进制字符串转换为16进制

    public static byte[] hexStringToByteArray(String hexString) {
    if (hexString == null || hexString.equals("")) {
        return null;
    }
    hexString = hexString.toUpperCase();
    int length = hexString.length() / 2;
    char[] hexChars = hexString.toCharArray();
    byte[] d = new byte[length];
    for (int i = 0; i < length; i++) {
        int pos = i * 2;
        d[i] = (byte) (charToByte(hexChars[pos]) << 4 | charToByte(hexChars[pos + 1]));
    }
    return d;
}

    byte数组转换为16进制字符串

    public static String bytesToString(byte[] bytes) {
    StringBuffer sBuffer = new StringBuffer();
    for (int i = 0; i < bytes.length; i++) {
        String s = Integer.toHexString(bytes[i] & 0xff);
        if (s.length() < 2) {
            sBuffer.append('0');
        }
        sBuffer.append(s + "");
    }
    return sBuffer.toString().toUpperCase();
}
